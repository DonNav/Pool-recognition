# Pool recognition

## Install dependencies 
Create a python venv:  
```    python -m venv ./  ```
Install required dependencies :
``` ./bin/pip3 install face_recognition numpy``` 
And for RaspberryPi :
``` ./bin/pip3 install picamera```

Or for standard webcam : 
``` ./bin/pip3 install opencv-contrib-python```

And clone the code :
``` git clone ```
